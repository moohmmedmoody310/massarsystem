# 🎓 مركز جمال ناصر التعليمي - نظام الإدارة المتكامل

## المميزات الكاملة للنظام

### 👥 إدارة الطلاب
- إضافة وتعديل وحذف الطلاب
- بيانات شاملة: الاسم، الهاتف، ولي الأمر، المرحلة، العنوان، الرقم القومي
- رقم واتساب ولي الأمر للتواصل الفوري
- عرض كشف حساب كامل لكل طالب
- سجل الحضور والغياب
- سجل النتائج والدرجات
- استيراد وتصدير بيانات الطلاب من/إلى Excel

### 👨‍🏫 إدارة المعلمين
- بيانات المعلمين الكاملة
- الراتب الأساسي وسجل صرف الرواتب
- عدد المجموعات والطلاب لكل معلم
- التواصل عبر واتساب

### 📚 إدارة المجموعات
- إنشاء مجموعات بمعلم ومادة ومرحلة محددة
- تحديد مواعيد الحصص وأيام الأسبوع
- الرسوم الشهرية لكل مجموعة
- الحد الأقصى لعدد الطلاب

### 💰 النظام المالي
- تسجيل المدفوعات الشهرية
- طرق دفع متعددة (نقدي، تحويل، فودافون كاش، إنستاباي)
- إيصالات PDF احترافية فورية
- تقرير شهري شامل PDF
- تصدير المدفوعات لـ Excel
- متابعة من لم يدفع بعينه
- إجمالي الإيرادات والمصروفات والصافي

### ✅ الحضور والغياب
- تسجيل يومي سريع لكل مجموعة
- سجل تاريخي كامل لكل طالب

### 📱 ربط واتساب
- إرسال تذكير بالدفع لولي الأمر مباشرة
- إرسال تذكير جماعي لكل من لم يدفع
- التواصل مع المعلمين عبر واتساب

### 📊 التقارير
- تقرير مدفوعات شهري PDF
- قائمة الطلاب Excel
- تقرير مالي Excel
- لوحة تحكم مع رسوم بيانية

## 🚀 التثبيت والتشغيل

### المتطلبات
- Windows 7 أو أحدث
- Python 3.8 أو أحدث
- اتصال إنترنت (لتثبيت المكتبات لأول مرة فقط)

### خطوات التثبيت

**الخطوة 1:** تأكد من تثبيت Python
- حمله من: https://www.python.org/downloads/
- **مهم:** ضع علامة على "Add Python to PATH"

**الخطوة 2:** افتح مجلد البرنامج

**الخطوة 3:** انقر نقراً مزدوجاً على `install.bat`
- سيقوم تلقائياً بتثبيت كل المكتبات
- وإنشاء قاعدة البيانات
- وإضافة اختصار على سطح المكتب

**الخطوة 4:** شغّل النظام
- انقر على أيقونة "مركز جمال ناصر" على سطح المكتب
- أو افتح `run_center.bat`
- سيفتح المتصفح تلقائياً

### بيانات الدخول الافتراضية
```
اسم المستخدم: admin
كلمة المرور: admin123
```

## 📁 هيكل الملفات

```
center_system/
├── app.py              ← السيرفر الرئيسي
├── database.py         ← قاعدة البيانات
├── pdf_generator.py    ← توليد التقارير PDF
├── install.bat         ← ملف التثبيت
├── run_center.bat      ← ملف التشغيل
├── templates/
│   ├── login.html      ← صفحة تسجيل الدخول
│   └── index.html      ← الواجهة الرئيسية
├── database/
│   └── center.db       ← قاعدة البيانات (تُنشأ تلقائياً)
└── reports/            ← مجلد التقارير
```

## 💡 نصائح الاستخدام

1. **النسخ الاحتياطي:** انسخ مجلد `database` بانتظام
2. **واتساب:** استخدم رمز الدولة (مثال: 201xxxxxxxxx لمصر)
3. **Excel:** يمكن تعديل البيانات في Excel ثم استيرادها
4. **الإيصالات:** تُفتح في المتصفح وتطبع مباشرة
